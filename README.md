# Quick Installation

1. Clone this repo to your `~` or `/home/username`path

```sh
git clone https://github.com/ad1822/hyprdots
```

2. Navigate to it

```sh
cd ~/hyprdots
```

3. Run this installer

```sh
sudo bash ./setup.sh
```
